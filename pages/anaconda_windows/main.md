# AnacondaをWindowsにインストールする手順

（2018年2月25日現在）

ある本で参照するためのものです。情報が古くなっていたら教えていただければと。

## 1. Anacondaのサイトに行く

* https://www.anaconda.com/

こんな画面が出ます。右上に「Download」というボタンがあるのでこれを押します。

<img src="anaconda_edge.png" width="400" />

ダウンロードページに飛ぶので、少しスクロールし、「Anaconda (バージョン） For Windows Installer」という文言を確認し、左下のPython 3.x用の「Download」ボタンを押し、インストーラをダウンロードします。
